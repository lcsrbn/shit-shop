# shit-shop

Next.js + Stripe + Vercel shop project for:

**https://idontneedthisshitandneitherdoyou.com**

## Stack

- Next.js 16
- React
- Stripe Checkout
- Vercel
- Namecheap DNS
- GitHub for version control

---

## Project naming

This project uses the same naming everywhere:

- Local folder: `shit-shop`
- GitHub repo: `shit-shop`
- Vercel project: `shit-shop`
- Docker container: `shit-shop-nextjs`
- Public domain: `idontneedthisshitandneitherdoyou.com`
